Στον φάκελο Experiments, υπάρχουν τα Measurements από το robot επεξεργασμένα. Ο σημαντικός φάκελος είναι ο Processed_Measurements όπου εκεί θα χρειαστεί να το τρέξεις όλο για να είσαι
σίγουρος ότι έχεις τα σωστά δεδομένα.

Το αρχείο Visualize_Data είναι μόνο για μερικά σχέδια. Το σημαντικό !! Processed_Measurements !!

Στο Graphs_For_Report έχει μόνο σχεδιάκια.


Στο Neural_Networks βρίσκονται όλα τα σκριπτάκια για τα νευρωνικά. Στο Dump Folder είναι όσα δεν χρησιμοποιούμε πλέον απλά τα ρίξαμε εκεί για να υπάρχουν σε περίπτωση ανάγκης.
Όλο το ζουμί είναι στο Straight_path. Έξω από τον φάκελο υπάρχουν τα σκριπτάκια για τις προσομοιώσεις τόσο σε 2 όσο και σε 3 διαστάσεις και τα σκριπτάκια για τα Σιαμαία δίκτυα τα 
οποία δεν πολυ-απέδωσαν.
Μέσα στο YesOffset υπάρχουν τα σκριπτάκια για τις 3 διαστάσεις όπου επιχειρήσαμε να βάλουμε offset στον άξονα Ζ- δεν δούλεψε καλά. 
Στο NoOffset είναι όλα τα "καλά" σκριπτάκια. Όποιο αρχείο γράφει ALLPERMS είναι για 3D και ΔΕΝ χωρίζει καλά τα δεδομένα οδηγώντας σε OVERFITTING - οπότε δεν είναι ΚΑΛΟ.
Το σκέτο PERMS είναι τα σωστά. Δηλαδή :
2D_WITH_CV -> 2D ΠΡΌΒΛΗΜΑ
3D_2_ANT_PERMS_CORRECT_LOGIC -> 3D, 2 ANTENNAS
3D_3_ANT_PERMS_CORRECT_LOGIC -> 3D, 3 ANTENNAS
QUADRUPLE_ANTENNAS -> 3D, 4 ANTENNAS
